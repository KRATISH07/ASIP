# ASIP Demo Sandbox

This directory contains the zero-database, ultra-low resource static sandbox edition of **ASIP (Autonomous Society Infrastructure Platform)**.

## How to Run Demo Sandbox
```bash
python3 run_asip.py
```
This runs the pre-compiled Next.js sandbox portal on `http://localhost:3000` using <25MB RAM, 0% CPU, and 0 database dependencies.
